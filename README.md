# AppleDHD

A beautifully minimalist, ADHD-focused Progress Tracker and No-Noise Calendar.

## Features
- **Job Tracker**: Track the days you've worked towards your monthly goal.
- **Study Tracker**: Track the exact hours you've studied against your monthly target.
- **No-Noise Calendar**: A clean, distraction-free calendar to track upcoming tests and deadlines.
- **AI Assistant**: A built-in minimalist AI to query your stats and upcoming events.

## Tech Stack
- React
- TypeScript
- Vite
- Lucide React (Icons)
- Vercel (Deployment)

## Getting Started
1. Run `npm install`
2. Run `npm run dev`
3. Open `http://localhost:5173`

*(Note: Designed to be installed as an iOS/Android PWA for a fully native experience!)*
